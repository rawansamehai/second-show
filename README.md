# Second Show | عرض تاني - Egyptian Art Heritage Platform

Second Show is a modern, bilingual (Arabic/English) art history platform dedicated to documenting and showcasing the rich heritage of Egyptian cinema, music, and visual arts.

## 🚀 Key Features

- **Bilingual Support**: Full Arabic and English support with automatic RTL/LTR layout switching.
- **Dark/Light Mode**: Seamlessly transition between cinematic dark and crisp light themes.
- **Progressive Web App (PWA)**: Installable as a standalone app on Windows, macOS, Android, and iOS with professional branding.
- **Interactive Experience**: Timeline of historical eras, filtered galleries, and interactive game sections.
- **Responsive Design**: Optimized for all devices, from mobile phones to high-resolution desktops.

## 🛠 Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/) (HashRouter)
- **Styling**: Vanilla CSS with custom properties for theme management.
- **Icons**: [Font Awesome](https://fontawesome.com/) & custom SVG assets.

## 📂 Project Structure

- `src/pages/`: Page-level components (Home, Music, Cinema, Films, Games, About).
- `src/components/`: Reusable UI elements like Navbar, Footer, and ScrollToTop.
- `src/utils/`: Translation data and helper utilities.
- `public/`: Static assets, PWA manifest (`site.webmanifest`), and Service Worker.

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/second-show.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
